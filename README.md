# Assignment 001

A simple Node.js server built with Express that returns my details in JSON format.

## Requirements

- Node.js installed
- npm

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/assignment-001.git
```

2. Navigate into the project

```bash
cd assignment-001
```

3. Install dependencies

```bash
npm install
```

## Running the Server

Start the server with:

```bash
npm start
```

The server will run at:

```
http://localhost:3000
```

  "role": "SIWES Student",
  "work": "Assignment 001"
